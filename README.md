# 🚀 NextHire - Frontend

**NextHire** is a modern job portal web application built with React, Tailwind CSS, and DaisyUI. This is the frontend of the full-stack NextHire project, designed to connect job seekers with employers through a clean, responsive, and user-friendly interface.

---

## 🖥️ Tech Stack

- **React**
- **React Router DOM**
- **Tailwind CSS**
- **DaisyUI**
- **Firebase Authentication**
- **Vite**

---

## 📸 Features

- 🔐 User & Admin Authentication (via Firebase)
- 🧑‍💼 Role-based Dashboard (User & Admin)
- 💼 Job Listings & Applications
- ✅ Private Route Protection
- 🎨 Responsive UI with Tailwind + DaisyUI
- ⚡ Fast performance with Vite
